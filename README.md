# Jogo da Velha (Tic-Tac-Toe) - React

Projeto desenvolvido utilizando a biblioteca **React** e estilizado com **Bootstrap**.

## Funcionalidades
- Tabuleiro interativo.
- Histórico de jogadas (Time travel).
- Indicação de vencedor ou empate.

## Como rodar o projeto
Como este projeto foi criado baseando-se no `create-react-app`, você pode rodar localmente com:

1. Instale as dependências:
   ```bash
   npm install
   npm start
